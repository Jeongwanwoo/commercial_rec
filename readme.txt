1. Topic
서울시 상권 추정매출, 유동인구, 상주인구 데이터를 활용하여 창업인에게 업종을 추천

2. Dataset
- 서울특별시_우리마을가게 상권분석서비스(상권_추정매출)
- 서울특별시_우리마을가게 상권분석서비스(상권_추정유동인구)
- 서울특별시_우리마을가게 상권분석서비스(상권_상주인구)

3. Start
service_rec.py 파일 경로 설정 -> run.py 파일 실행 -> 터미널창의 경로 이용하여 시스템 이용 

4. Result
Top5 매출 업종, Top5 밀집도별 업종을 추천함과 동시에 로드뷰와 지도로 상권의 위치를 알려준다.
